Datenbank

04.10.2018

Ihre Daten liegen in CouchDB.

Die herausragende Spezialität von CouchDB ist Synchronisieren.
Keine andere Datenbank kann das so gut.

Um CouchDB herum ist eine ganzes Ökosystem von Werkzeugen entstanden.
Zwei davon werden von erfassen.ch verwendet:

  • PouchDB: JavaScript-Version von CouchDB
  • RxDB: Baut auf PouchDB auf und macht es reaktiv
